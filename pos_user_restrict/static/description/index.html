Restriction of POS User

The module allows to set the allowed Points of Sale for POS users and restricts access for other Points.

Module Features

Restricts access to Points of Sale
Simply Configurable
  • Restrictions work only for users with role Point of Sale / User.
  • After configuration, users will be able to see only the allowed Points of Sale and POS orders related to these Points.
  • Users with the role Point of Sale / Manager have access to all Points of Sale and POS orders.
  • Configured from the User Settings form.
