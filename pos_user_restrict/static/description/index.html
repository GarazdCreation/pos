Restriction of POS User

The module allows to set the allowed Points of Sale for users and restricts access for other Points.

Module Features

Restrictions work only for users with role "Point of Sale / User".
After configuration users will be able to see only the allowed Points of Sale, POS orders and payments related to these Points.
Users with the role "Point of Sale / Manager" have full access to all Points of Sale, POS orders and payments.
Simply configurable from the user settings form.

Configuration

To configure please follow these steps:

  • Go to the Settings - Users menu.
  • Set up the Allowed POS in the user form.
  • Changes will be applied immediately after saving.
